resizer-and-converter

Código em Python que permite redimensionar e converter imagens de forma simples e eficiente.

O redimensionamento ajusta automaticamente a imagem para a resolução de 1200x1600, reduzindo drasticamente o tamanho do arquivo, mantendo o aspect ratio original e preservando a qualidade da imagem.

A conversão permite transformar imagens nos formatos .HEIC, .PNG e .WEBP para o formato .JPG.

O projeto pode ser executado de duas formas:

- Localmente, com uma interface gráfica simples desenvolvida com customtkinter.

- Pelo navegador, utilizando streamlit. Link para uso: https://resizer-and-converter.streamlit.app/

Tecnologias utilizadas:

- Python

- Pillow

- customtkinter

- streamlit
